## 怎么用？三步搞定

### 第1步：把组件放到公网

组件是 HTML 文件，Notion 需要公网链接才能嵌入。推荐 GitHub Pages（免费）：

1. 注册 GitHub → 新建仓库 → 上传组件文件
2. Settings → Pages → Branch 选 main → Save
3. 等 30 秒，得到链接：`https://你的用户名.github.io/仓库名/components/日历.html`

### 第2步：在 Notion 里嵌入

1. 打开你的 Notion 页面
2. 输入 `/embed`
3. 粘贴组件的公网链接
4. 点 Embed → 完成

### 第3步：调整大小

拖拽组件右下角调整。推荐宽度：
- 日历：300-350px
- 番茄钟：200-250px
- 倒计时：280-350px
- 导航栏：全宽

## 常见问题

**Q: 组件会泄露我的 Notion 内容吗？**
A: 不会。组件是纯 HTML/CSS/JS，在你自己 Notion 里用 iframe 渲染，不读取 Notion 数据。

**Q: 手机能用吗？**
A: 全部组件响应式设计，手机端自适应。

**Q: 能改颜色吗？**
A: 标准版和 Pro 版可以。联系元五七解锁后获得完整源文件，改 CSS 变量即可。

**Q: 为什么免费版只有 10 个？**
A: 免费版让你先体验。好用再升级标准版/Pro版，私信元五七解锁。

**Q: 买完怎么收到货？**
A: 私信元五七 → 付款 → 收到压缩包下载链接。解压 → 上传到 GitHub Pages → 嵌入 Notion。
